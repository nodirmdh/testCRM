# LMS для оффлайн учебных центров

B2B SaaS система управления учебными центрами с подписочной моделью.
Фокус - операционное управление: ученики, группы, финансы, зарплаты и сертификаты.

## Ключевые возможности
- управление учениками, родителями и преподавателями
- группы, курсы, расписание, посещаемость
- финансы: счета, оплаты, доходы
- зарплаты преподавателей (за занятие или % от дохода)
- выдача сертификатов

## Правила работы с проектом
- Сначала уточняем требования и план, затем реализуем.
- Любые изменения требований фиксируем в документации.
- Не усложняем архитектуру без необходимости.
- Все сущности учитывают multi-tenant через `organization_id`.
